# AI-Powered Resume Screening and ATS Scoring System

## Project Overview

This project is an AI-based Resume Screening and ATS Scoring System that extracts skills from uploaded PDF resumes and evaluates candidate suitability based on job descriptions using NLP and BERT semantic similarity.

The system automates resume evaluation and helps shortlist candidates efficiently.

---

## Features

- PDF Resume Upload
- Resume Text Extraction
- NLP Text Cleaning
- Automatic Skill Extraction
- ATS Score Calculation
- BERT Semantic Similarity Matching
- Resume Ranking

---

## Technologies Used

- Python
- NLTK
- Scikit-learn
- Sentence Transformers
- BERT
- PyPDF2
- Jupyter Notebook
